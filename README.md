# 🛍️ Nostra – Fashion E-Commerce Website

A modern and responsive fashion e-commerce website designed to showcase stylish collections with a clean UI and seamless user experience.

---

## 🚀 Features

- 🛒 Browse fashion products with clean UI  
- 🔍 Product listing and category sections  
- ❤️ Attractive and modern design  
- 📱 Fully responsive (mobile, tablet, desktop)  
- ⚡ Fast and smooth user experience  
- 🎯 User-friendly navigation  

---

## 🧰 Tech Stack

- 🌐 HTML5  
- 🎨 CSS3 / Tailwind CSS  
- ⚡ JavaScript (if used)  
- ☁️ Deployment: Vercel
